k-NN for Heights and Weights

The "k-NN for Heights and Weights" project is designed to predict the sex of a person based on their height and weight. It utilizes the k-nearest neighbors (k-NN) algorithm, which is a non-parametric classification algorithm commonly used in machine learning. The project was implemented using a Conda environment, which provides a controlled and isolated environment for running the code. The data used for training and testing the model was obtained from a survey that collected information on height, weight, and sex.

k-means for Microprocessors

The "k-means for Microprocessors" project focuses on creating a model that can provide an approximate price for a microprocessor based on its frequency. It utilizes the k-means clustering algorithm, which is a popular unsupervised machine learning algorithm used for grouping similar data points. The project starts by collecting data from a list of microprocessors, including their price and frequency (in GHz). The dataset serves as the basis for training the k-means model. The algorithm groups the microprocessors into clusters based on their frequency, aiming to identify patterns and similarities among them. By analyzing the clustered data, the project aims to uncover relationships between the frequency and price of microprocessors. This information can then be used to create a model that can estimate the price of a microprocessor based on its frequency. The model learns from the patterns observed in the training data and applies them to make predictions for new, unseen microprocessors.

05.

Where is my pozole?

"Where is my Pozole?" is a puzzle solver project that aims to find the best path for an agent represented in an environment with obstacles to reach its goal without encountering any obstacles along the way. The project utilizes various search techniques, including best-first search, A-star search, and hill climbing search, to navigate through the puzzle and determine the optimal path for the agent. The environment is represented as a grid, where each cell can either be an obstacle or an open space. The agent's objective is to reach the goal cell, which contains the cherished Pozole, a traditional Mexican dish. The agent's movement is constrained by the presence of obstacles, and it must find the safest and most efficient route to its destination.

Medication for the correct patient

"Medication for the Correct Patient" is a project that utilizes a decision tree algorithm to predict the most suitable medication for a patient based on various input data, including age, sex, blood pressure, and cholesterol levels. The goal of this project is to assist healthcare professionals in making informed decisions about prescribing medications tailored to individual patients' characteristics. The decision tree algorithm analyzes the provided data and creates a hierarchical structure of decision nodes, where each node represents a feature or attribute. The algorithm makes splits in the tree based on the most discriminative features, creating branches that lead to different medication recommendations. The decision nodes contain conditions that determine which path to follow based on the patient's data.

07.

TicTacToe

"TicTacToe" is a project that implements the popular game of Tic-Tac-Toe using the minimax algorithm. The game allows two players to play against each other, with one player being a human and the other being the computer. The minimax algorithm is a recursive algorithm commonly used in game theory and decision-making. In the context of Tic-Tac-Toe, it helps the computer player make optimal moves by considering all possible future moves and their outcomes.
